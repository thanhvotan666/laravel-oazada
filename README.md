
# 🛒 Oazada – E-Commerce Website with Realtime Chat (Laravel MVC)

An e-commerce web application built with Laravel MVC, featuring real-time chat functionality using Pusher, enabling seamless communication between buyers and sellers.

## 📌 Features

- User registration and authentication
- Role-based access control (Admin – Seller – Buyer)
- Product, category, and inventory management
- Shopping cart and order management
- Real-time chat between buyers and sellers (Pusher)
- Product search, filtering, and pagination
- Review and rating system
- Order status tracking
- Notification system for new orders
- Secure form validation and middleware protection
- Responsive UI design

## 🛠️ Technologies Used

- Backend: PHP, Laravel (MVC Architecture)
- Frontend: Blade Template, HTML, CSS, JavaScript, AJAX, Bootstrap
- Database: MySQL
- Realtime: Pusher WebSocket
- Tools: Git, GitHub, VS Code, Postman
